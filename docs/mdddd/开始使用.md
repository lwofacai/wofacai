## 🛠️ 第一步：安装「沉浸式翻译」插件（强烈推荐）

本站的部分核心文档源自互联网最前沿的英文开源社区。为了打破语言障碍，同时保留原文的技术语境，我们推荐所有用户安装 **[沉浸式翻译]** 插件。

### 为什么选择它？
* **双语对照**：它不会直接替换原文，而是将中文显示在英文下方，方便你核对专业术语或原始链接。
* **无缝体验**：对于本站正在进行汉化的文档，或者通过本站跳转到的外部英文站点，它能让你实现“无痛”阅读。

### 安装步骤

1.  **下载插件**：
    * **Chrome / Edge 浏览器**：前往 [Chrome 应用商店](https://chrome.google.com/webstore/detail/immersive-translate/bpoadfkcbjbfhfodiogcnhhhpibjhbnh) 搜索 "Immersive Translate" 并点击「添加至 Chrome」。
    * **Firefox 浏览器**：前往 [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/immersive-translate/) 下载。
    * **Safari / iOS**：在 App Store 搜索下载。

2.  **开启翻译**：
    安装完成后，当你访问英文网页时，点击浏览器右上角的插件图标，或者直接按快捷键（通常是 `Alt + A`），页面即刻变为中英对照模式。
